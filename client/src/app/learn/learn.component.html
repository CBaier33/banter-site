Fundamental Syntax

If you're already familiar with programming, you might find Banter mildly infuriating. There are no loops, no functions, and only primitive data types. The whole language can be broken down into five basic rules. While this may be limiting for building enterprise software, it is actually quite liberating for the mind of a student.

The Let Statement

allows you to assign the output of an expression to a particular mnemonic representation.

let x be 5

The If Statement

allows you to conditionally execute a line of code. The body of the statement will only execute if the expression evaluates to True

if x < 10, then
  let y be 10

The If-Else Statement

simply provides alternative execution case for when the condition evaluates to False.

if x * y == (100/2) - 1, then
  let x be False
else
  let x be True

The Return Statement

is used to simply return the value of a statement or expression and terminate the evaluation. These statements cannot be null, you must return something.

let wrong be 3
let right be 7

if wrong + wrong == right, then
  return True
else
  return False

return "this will never execute"

The Goto and Marker Statements

allow for repetition of instructions by returning back to demarcated lines of code.

let x be 0
let n be 100

let a be 0
let b be 1

@1 # Print Fibonacci Numbers
if x < n, then
  print a

  let temp be a + b
  let a be b
  let b be temp


  let x be x + 1

  goto instruction 1

return "This is a valid result"
